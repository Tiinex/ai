# Operational Addendum Template v1

Use this only when the receiving instance should move from transfer discipline into concrete work.

This addendum is intentionally small. It should complement the transfer artifact, not replace it.

## Purpose

Add only the minimum project-specific context needed for safe operational behavior.

## What This Adds

- the concrete task or decision currently in scope
- the minimum canon required to act safely
- the current working state that cannot be guessed
- any environment or policy constraints that block normal progression
